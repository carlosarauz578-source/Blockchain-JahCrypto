# 📖 Bitácora JahCrypto – Bloque 0

## Reflexiones
- La disciplina y la organización son tan importantes como el conocimiento técnico.  
- Crear una identidad propia (*JahCrypto*) me ayuda a diferenciar mi portafolio y darle coherencia.  
- Publicar semanalmente no solo refuerza mi aprendizaje, también construye visibilidad profesional.  
- El estudio autónomo requiere constancia: pequeños avances diarios son más efectivos que sesiones largas y esporádicas.  

## Aprendizajes
- Comprendí la estructura del programa y cómo cada bloque se conecta con el siguiente.  
- Aprendí a configurar mi entorno de trabajo en Termux y GitHub para documentar el progreso.  
- Reconocí la importancia de usar flashcards y checklists como herramientas de memoria y organización.  
- Entendí que cada bloque debe terminar con un mini-proyecto y una publicación, para consolidar teoría y práctica.
